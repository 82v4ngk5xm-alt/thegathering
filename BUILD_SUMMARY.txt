üìä DATABASE SCHEMA
================================================================================

Tables Created:
  1. scriptures
     - id (UUID)
     - book (TEXT)
     - chapter (INTEGER)
     - verses (TEXT)
     - text (TEXT)
     - translation (TEXT)
     - display_order (INTEGER)
     - background_image_url (TEXT)
     - created_at (TIMESTAMP)
     - updated_at (TIMESTAMP)

  2. comments
     - id (UUID)
     - scripture_id (UUID) [Foreign Key]
     - author_name (TEXT)
     - author_email (TEXT)
     - text (TEXT)
     - is_approved (BOOLEAN)
     - created_at (TIMESTAMP)

Indexes Created:
  - idx_scriptures_display_order
  - idx_comments_scripture_id
  - idx_comments_is_approved

Row Level Security (RLS):
  - Public can read scriptures
  - Public can read approved comments
  - Anyone can insert comments
  - Service role can modify all data
